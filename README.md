# Парикмахерская на Laravel

Лендинг для барбершопа с возможностью онлайн-записи.
Технологии
- Laravel 10
- Bootstrap 5
- MySQL

Как запустить
1. Склонируйте репозиторий:
git clone https://github.com/Markus0896/barbershop-laravel.git

2. Установите зависимости:

composer install

3. Запустите сервер:

php artisan serve
